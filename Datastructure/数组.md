## 定义简述
数组为最基础的数据结构, 因为其定义就是贴近与内存当中的地址分布，定义一个数组即为申请一块连续的内存空间。

```java
int[] arr = new int[10];
```
两个基本要素:  引用(指针、起始地址值) + 索引范围
例如这里 定义了 一个int数组，int[10]就意味着支持容纳10个int值对象, 索引值为0~9。
编译器(?)可以通过 arr (起始地址) + index (索引值) * sizeofArrayObject(数组对象的字节大小, 例如int值为4个字节) 获取到对应对象内容


## 特点

- 查询

   根据索引值就可以快速定位, 复杂度为 O(1)
   ```java
      int i = arr[0];
   ```

- 更新

   同样根据索引位置可以快速定位到需要更新的位置, 复杂度为 O(1)
   ```java
       arr[3] = 4;
   ```

- 新增
  
  末尾新增, 只需要按照索引值将内容设置进去即可, 与查询类似，只是读改为写, 复杂度为O(1)

  但, 如果需要在中间位置插入某个值, 就需要设计到 数据搬迁
  例如   1 | 2 | 3 | 4 | 5 ， 需要在3和4之间插入一个6, 不可以直接将4给覆盖为6，这样4的数据就会丢失, 因此需要将 3 后一位的索引对应的数据都往后移1位

  ```java
     // 初始化 5位数组
     int arr[10];
     for (int i = 0; i < 5; i++) {
      arr[i] = i + 1;
     }

     // 需要从index = 3 当中插入一个 num = 6

     for (int i = 5; i > 3; i-- ) {
       arr[i] = arr[i - 1]
     }

     arr[3] = 6
  ```


总结模板

```java
// 原有数组长度 size
// 需要插入的索引位 index

for (int i = size; i > index ; i--) {
   arr[i] = arr[i - 1]
}

arr[index] = num
```
